Project one LBTT Tax Calculator

Reverse engineered website

The Task: Buying a house in the UK incurs a tax which varies according to the price of the house and circumstances of the purchaser e.g., a higher rate is paid if you are buying a second home.

This website was used as a reference:

I was working with Python and PyScript, which I hadn't used beforehand. For this project I was using Unit tests and it was the first time I implemented them to test my code. The project was done in a more practical way not focusing on looks, just functionality and made using Html.

During my code implementing I ran into a problem, where my source template calculations were different to my project. First I tried to understand the problem and define it. Afterwards I searched for solutions online and systematically went through them one by one, noting what I had done. Having done all that, I retraced my steps and before I could ask my developer friend for advice telling him what I had done, how that had resulted and what was the current issue. I noticed a slight issue with numbers in code that I fixed which resulted in the expected outcome.
