Project one LBTT Tax Calculator

HTML, Pyscript and Python

The Task: Buying a house in the UK incurs a tax which varies according to the price of the house and circumstances of the purchaser e.g., a higher rate is paid if you are buying a second home.

This website was used as a reference:

I was working with Python and PyScript, which I hadn't used beforehand. For this project I was using Unit tests and it was the first time I implemented them to test my code. The project was done in a more practical way not focusing on looks, just functionality and made using HTML.

Calculator

This was a difficult issue to query on Google as there were no error messages to search for. I ultimately fell back on the test driven approach. The logic was defined on the task, so I tried writing tests that cover all aspects of it. One of the tests failed which helped me identify where the mistake was made in the code. The error ended up being a typo, but it helped me learn in practice the value of writing unit tests!
